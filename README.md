# 🍔 Evento NLW Experts - BurgerApp

**BurgerApp** é um aplicativo desenvolvido durante o evento NLW Experts, projetado para oferecer uma experiência completa de criação de hambúrgueres personalizados. Com ele, você pode montar o seu hambúrguer ideal, escolher entre diversos ingredientes e acompanhar o status do seu pedido em tempo real.

## 🚀 Como baixar e instalar o aplicativo

Você pode baixar o aplicativo através do seguinte link:  
[Download BurgerApp](https://drive.google.com/file/d/1ipPBkmtf_0BtZTyDvzxLqFqfYWDdkmtP/view)

## 🛠️ Tecnologias Utilizadas

O aplicativo foi construído utilizando as seguintes tecnologias:

- **TypeScript**
- **JavaScript**
- **React Native**

## 📱 Como rodar o projeto localmente

1. Certifique-se de ter o [Expo](https://expo.dev/) instalado em sua máquina.
2. Clone este repositório.
3. Na pasta do projeto, execute o seguinte comando para iniciar o aplicativo:

   ```bash
   npx expo start --clear
   ```

4. Para rodar o aplicativo, você precisará ter o aplicativo **Expo Go** instalado em seu celular (Android ou iOS), ou ter o **Android Studio** configurado em seu computador. O ambiente funciona tanto no **Windows**, **MacOS** quanto no **Linux**.

## 📦 Funcionalidades do aplicativo

- **Montar Hambúrgueres**: Personalize seu hambúrguer escolhendo entre diversos ingredientes.
- **Acompanhamento de Pedido**: Veja o status do seu pedido em tempo real.
- **Histórico de Pedidos**: Acesse seus pedidos anteriores e peça novamente com um clique.
- **Notificações**: Receba atualizações sobre a entrega diretamente no seu celular.

## 💻 Requisitos

- Node.js
- Expo CLI
- Android Studio (se for testar em um emulador)
- Expo Go (se for testar no celular)
